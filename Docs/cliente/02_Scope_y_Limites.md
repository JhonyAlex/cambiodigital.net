# Alcance y Límites del Servicio: IT Partner

En Cambiodigital, creemos que la claridad es la base de una gran relación comercial. Este documento detalla con precisión qué tareas cubre tu suscripción a IT Partner y cuáles se consideran un trabajo aparte. Nuestro objetivo es que siempre sepas qué esperar de nuestro servicio.

## La Diferencia Fundamental: Soporte vs. Proyecto

Para entender el alcance, es crucial distinguir entre dos tipos de trabajo: **Soporte** y **Proyecto**.

-   **Soporte (✅ Incluido):** Se enfoca en **mantener, optimizar y reparar** la infraestructura y sistemas que ya existen. Son tareas recurrentes y reactivas diseñadas para garantizar la estabilidad y el buen funcionamiento de tu operación actual. El soporte está cubierto por tu suscripción mensual.

-   **Proyecto (❌ Excluido):** Implica **crear, construir o transformar** algo fundamentalmente nuevo. Son trabajos con un principio y un fin definidos, que resultan en una nueva capacidad o un cambio estructural en tu infraestructura. Los proyectos se cotizan de forma independiente a tu suscripción.

**Nuestra promesa:** Si una solicitud tuya califica como un "Proyecto", nuestro equipo te lo notificará **antes** de realizar cualquier trabajo. Te explicaremos por qué se considera un proyecto y te presentaremos una cotización detallada para tu aprobación. **Nunca recibirás una factura por un trabajo que no hayas aprobado previamente.**

## ✅ Tareas Incluidas en tu Suscripción (Ejemplos de Soporte)

Las siguientes actividades están cubiertas por tu plan mensual de IT Partner. El objetivo es mantener tu sistema funcionando sin problemas.

| Categoría | Ejemplos Concretos de Tareas Incluidas |
| :--- | :--- |
| **Resolución de Incidentes** | - Tu sitio web está caído o muestra un error 500.<br>- El servidor está funcionando muy lento o no responde.<br>- Una automatización clave (ej. Zapier) ha dejado de funcionar.<br>- Los usuarios no pueden iniciar sesión en una de tus plataformas.<br>- Se ha detectado un problema con la copia de seguridad nocturna. |
| **Ajustes y Configuraciones Rápidas** | - Cambiar la configuración de un plugin de WordPress.<br>- Actualizar las credenciales de acceso a un servicio.<br>- Reinstalar un módulo que está fallando.<br>- Ajustar una regla en tu firewall.<br>- Modificar una automatización existente que no funciona como se esperaba. |
| **Mantenimiento Preventivo** | - Aplicar actualizaciones de seguridad a tu CMS (WordPress, etc.).<br>- Actualizar plugins y temas a sus últimas versiones estables.<br>- Realizar pruebas de restauración de backups.<br>- Limpiar la base de datos de registros innecesarios.<br>- Monitorear el espacio en disco y el uso de recursos del servidor. |
| **Consultoría Operativa** | - "¿Cuál es la mejor manera de configurar este nuevo plugin?".<br>- "Estamos viendo lentitud en el sitio, ¿pueden diagnosticar la causa?".<br>- "¿Qué herramienta nos recomiendan para hacer X tarea?".<br>- "¿Es segura nuestra configuración actual?".<br>- Revisar un informe de rendimiento y proponer mejoras. |

### La Regla de las 2 Horas

Como guía general, la mayoría de las tareas de **ajuste y configuración** que toman **menos de 2 horas** de trabajo técnico se consideran parte del soporte incluido. Si una tarea de configuración supera este umbral, es probable que se clasifique como un mini-proyecto y podría requerir una cotización.

## ❌ Tareas Excluidas de la Suscripción (Ejemplos de Proyecto)

Las siguientes actividades no están cubiertas por tu plan mensual y requieren una cotización separada. Estos trabajos implican la creación de algo nuevo o cambios significativos en tu infraestructura.

| Categoría | Ejemplos Concretos de Tareas Excluidas |
| :--- | :--- |
| **Desarrollo de Software a Medida** | - Crear un nuevo plugin de WordPress desde cero.<br>- Desarrollar una aplicación web o móvil.<br>- Escribir un script complejo para una funcionalidad personalizada.<br>- Construir una integración a medida con una API de terceros. |
| **Migraciones Estructurales** | - Mover tu sitio web de un proveedor de hosting a otro.<br>- Migrar una base de datos completa a un nuevo sistema.<br>- Cambiar la arquitectura fundamental de tu aplicación.<br>- Realizar un despliegue completo de tu infraestructura en un nuevo entorno. |
| **Implementaciones Masivas** | - Configurar un CRM (como HubSpot o Salesforce) desde cero.<br>- Implementar un sistema ERP para toda la empresa.<br>- Desplegar una nueva plataforma de e-commerce completa.<br>- Crear un sistema de automatización de marketing con docenas de flujos. |
| **Trabajos de Diseño y Contenido** | - Diseño gráfico de banners o logos.<br>- Creación o edición de contenido para tu sitio web (redacción de artículos, etc.).<br>- Producción o edición de video. |
| **Mejoras de Gran Alcance** | - Rediseñar completamente el aspecto visual de tu sitio web.<br>- Añadir una nueva sección principal a tu aplicación que requiere desarrollo.<br>- Cualquier trabajo que se estime que tomará **más de 8 horas** de dedicación. |
